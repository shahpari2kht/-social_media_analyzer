# راهنمای مشارکت در پروژه

از شما خوشحالیم که می‌خواهید در پروژه `social_media_analyzer` مشارکت کنید!  
لطفاً قبل از شروع کار، این نکات را مطالعه کنید:

## قوانین کلی
- لطفاً پیش از ایجاد Issue، بررسی کنید که مورد شما قبلاً ثبت نشده باشد.
- برای Pull Request، از آخرین نسخه شاخهٔ اصلی پروژه استفاده کنید.
- Commit message‌ها باید کوتاه، واضح و حرفه‌ای باشند.

## ساختار Pull Request
- توضیح کامل تغییرات
- نوع تغییر (رفع باگ، افزودن ویژگی، بهبود مستندات و غیره)
- بررسی و تست کد

## گزارش مشکلات
برای گزارش مشکلات از قالب `bug_report.md` استفاده کنید.
برای پیشنهاد ویژگی یا بهبود، از قالب `feature_request.md` استفاده کنید.
